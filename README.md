# 🤖 RAG Chatbot

An AI-powered chatbot built using **Retrieval-Augmented Generation (RAG)** that provides context-aware responses by retrieving relevant information from a knowledge base and using an LLM to generate accurate answers.

## 🚀 Overview

Traditional chatbots rely only on the knowledge stored in the language model, which can lead to outdated or incorrect answers. This project uses **RAG architecture** to combine document retrieval with language generation.

The chatbot:
- Processes and stores documents as vector embeddings
- Retrieves relevant information based on user queries
- Uses an LLM to generate responses grounded in retrieved context


## ✨ Features

- 💬 Context-aware AI conversations
- 🔍 Semantic search using embeddings
- 📚 Document-based question answering
- 🧠 Retrieval-Augmented Generation pipeline
- ⚡ Faster and more accurate responses compared to a standalone LLM
- 🔄 Supports custom knowledge bases

## 🛠️ Tech Stack

- **Python**
- **LangChain** - RAG pipeline orchestration
- **ChromaDB** - Vector database
- **Hugging Face Embeddings** - Text embeddings
- **Large Language Model (LLM)** - Response generation
